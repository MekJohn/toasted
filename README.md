# tosted
Win toast notification package for python.
Build your own toast starting from a readable human xml class natively implemented.

## Installation

```bash
pip install toasted
```

## Usage

### Fast toast

```python
import toastes as ts

reminder_toast = ts.Reminder("Hi. Please call me")
reminder_toast.send()

```
